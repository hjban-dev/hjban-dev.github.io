# React에서 JSX 사용하기

## JSX?
- JavaScript + XML의 합성어
- 자바스크립트의 확장문법

### app.js를 JSX로 구성

```javascript
const title = <h1>My First React Element</h1>;
const desc = <p>This is Description</p>;
 
const header = React.createElement(
  'header',
  null,
  title, desc
);
 
ReactDOM.render(header, document.getElementById('root'))
```
